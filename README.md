# cargowise-angular-project

This is a frontend web project developed during my internship. The project is built using **Angular** and focuses on creating a user interface for a **CargoWise**-related logistics dashboard.

## 🚀 Features

- Responsive UI using **Bootstrap**  
- Structured layout with **Angular Components**  
- Clean design with **HTML & CSS**  
- Dynamic form handling and data display  
- Easy to integrate with backend APIs  

## 🔧 Tech Stack

- Angular  
- HTML5  
- CSS3  
- Bootstrap  

## 📁 Folder Structure

src/ ├── app/ │ ├── components/ │ ├── services/ │ └── ... ├── assets/ └── index.html

## 🛠 How to Run

1. Clone the repository  
   `git clone https://github.com/yourusername/cargowise-angular-project.git`

2. Navigate into the folder  
   `cd cargowise-angular-project`

3. Install dependencies  
   `npm install`

4. Run the project  
   `ng serve`

5. Open your browser and go to `http://localhost:4200/`
